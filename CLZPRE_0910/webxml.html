Kurzus Tematika

A kurzus a strukturált adatok weben történő tárolásával, továbbításával és feldolgozásával foglalkozik. Kiemelt figyelmet kapnak az adatcsere formátumok.

  • XML (eXtensible Markup Language) alapjai és szintaxisa.
  • XML sémák (DTD, XSD) definíciója.
  • XSLT transzformációk.
  • JSON (JavaScript Object Notation) felépítése és használata.
  • Adatküldés kliens és szerver között (AJAX alapok).
